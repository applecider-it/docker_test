## その他

reactはメンテナンスコストが高くなりがちなので非推奨。

このレポジトリ内にあるコードは、あくまで、実装例です。

なぜ、非推奨なのかは、実装例でVueと比較してあるので、見ればわかると思う。

reduxもグローバル変数になるためカプセル化できなくなるので、おすすめできません。

アトミックデザインはあくまで実装例で非推奨。

理由は、依存が複雑になって変更が危険になるためです。

個人的な趣味の範囲で実装すること自体は楽しいと思います。

ブロックチェーンは国際送金以外には、あまり使い道がない気がしますが一応動作確認をしています。

turboの実装例もありますが、全ページ常時WebSocket接続が必要な状況で、MPAにしたいとき以外は非推奨。

なぜ非推奨かは、実装例を見ればわかると思う。
